# csswx

#### 介绍
java开发的后台服务端，一个简单的SpringBoot框架，包含一些微信操作的小工具类，可以为公众号，小程序做后台用

小程序的项目为【wxcss】https://github.com/sunqiang10/wxcss
#### 软件架构
SpringBoot，maven，mysql

#### 使用说明
只需要修改application.properties配置，编写自己的Controller等即可
